# dynamical-resolve-ip

### 仅支持cloudflare

### 使用方法

1.在release中下载jar文件

2.使用Java环境启动

3.第一次打开会生成配置文件并关闭

配置文件如下:

    info:
        //cloudflare 邮箱
        email:
        //cloudflare Global API
        GKey:
        //目标域名,就是需要解析的域名,填完整如 www.example.com
        name:
        //解析类型
        type: A
